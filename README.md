## Description
This game is about how I personally make pancakes, which I love to do. In this game, you're given instructions on how to make the batter for pancakes, that is all you have to do, shouldn't be too difficult!
Click each ingredient listed on the instructions at the bottom right of the screen, if you click the wrong ingredient, you fail!
The src folder has the code in it, along with all of the pictures (the title, the background, the ingredients, ect.) and the proposal file. Thank you for playing!
